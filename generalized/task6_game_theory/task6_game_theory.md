# Задача 6. Матричная игра с нулевой суммой

## Тип задачи
Теория игр: антагонистическая игра двух лиц с нулевой суммой. Нужно найти
оптимальные смешанные стратегии обоих игроков и цену игры. Задача сводится к
паре двойственных задач линейного программирования, решаемых `linprog`
(метод HiGHS) из **SciPy**.

## Исходная постановка
Задана платёжная матрица 3×3 (выигрыши игрока A). Требуется определить
оптимальные смешанные стратегии игроков A и B и цену игры.

## Как обобщено
- Поддерживается **платёжная матрица произвольного размера** m×n (число
  стратегий игроков может различаться).
- Данные вынесены в `data/default.json`.
- Добавлен **сдвиг платёжной матрицы** на константу, если в ней есть
  отрицательные или нулевые элементы. Это гарантирует положительность цены во
  внутренней ЛП-формулировке и делает решение устойчивым при любых числах;
  итоговая цена игры восстанавливается обратным сдвигом.
- Стратегии обоих игроков считаются согласованно (цена игры v совпадает для
  прямой и двойственной задач).

## Специфичные настройки
- Отдельных флагов нет — поведение определяется данными.
- В данных:
  - `payoff` — платёжная матрица «стратегии A × стратегии B» (выигрыш A);
  - `row_strategies`, `col_strategies` — подписи стратегий;
  - `row_player`, `col_player` — имена игроков для вывода.

## Файлы данных
- `data/default.json` — исходная игра 3×3 со смешанным решением.
- `data/variation_saddle_point.json` — игра с седловой точкой (есть чистое
  решение): демонстрирует случай, когда оптимум достигается на чистых
  стратегиях.
- `data/variation_rectangular.json` — прямоугольная матрица 2×4: показывает
  работу с неквадратной игрой.

## Запуск
```bash
python task6_game_theory.py
python task6_game_theory.py --input data/variation_saddle_point.json
python task6_game_theory.py --input data/variation_rectangular.json
python task6_game_theory.py --interactive
```
