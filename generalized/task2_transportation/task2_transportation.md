# Задача 2. Транспортная задача

## Тип задачи
Линейное программирование. Нужно так распределить перевозки однородного груза
от поставщиков (складов) к потребителям (магазинам), чтобы суммарная стоимость
доставки была минимальной при соблюдении запасов и потребностей. Решение —
средствами **PuLP**.

## Исходная постановка
Два склада с заданными запасами снабжают три магазина с заданными
потребностями. Известна матрица стоимостей доставки «склад → магазин».
Требуется минимизировать общие затраты.

## Как обобщено
- Поддерживается **произвольное число складов и магазинов**.
- Все данные вынесены в `data/default.json`.
- Добавлена обработка **несбалансированной задачи**: если суммарный запас не
  равен суммарному спросу, соответствующая группа ограничений автоматически
  заменяется неравенствами (`<=`), и выводится предупреждение. Сбалансированная
  задача по-прежнему решается через равенства.
- Добавлена проверка согласованности размеров.

## Специфичные настройки
- `--integer` — искать целочисленный план перевозок.
- В данных:
  - `supply` / `demand` — запасы складов и потребности магазинов;
  - `cost` — матрица «склады × магазины»;
  - `currency` — подпись денежных единиц.

## Файлы данных
- `data/default.json` — исходная сбалансированная задача (2 склада, 3 магазина).
- `data/variation_unbalanced_surplus.json` — суммарный запас превышает спрос:
  демонстрирует работу с несбалансированной задачей и предупреждением.
- `data/variation_large_network.json` — сеть 3 завода × 4 региона: показывает
  масштабирование на большую размерность.

## Запуск
```bash
python task2_transportation.py
python task2_transportation.py --input data/variation_unbalanced_surplus.json
python task2_transportation.py --input data/variation_large_network.json --integer
python task2_transportation.py --interactive
```
