# Задача 5. Оптимизация инвестиционного портфеля

## Тип задачи
Задача квадратичного программирования (модель Марковица). Нужно подобрать доли
активов в портфеле, минимизирующие риск (стандартное отклонение доходности) при
условии полного вложения средств и заданных ограничениях на доли. Решение —
`minimize` (метод SLSQP) из **SciPy**.

## Исходная постановка
Два актива A и B с известными доходностями, волатильностями и корреляцией. Нужно
найти доли, дающие минимальный риск портфеля, при ограничении на максимальную
долю одного из активов.

## Как обобщено
- Модель расширена с двух активов до **произвольного числа активов**: вместо
  явной формулы дисперсии для двух бумаг строится ковариационная матрица из
  вектора волатильностей и матрицы корреляций, риск вычисляется как
  `wᵀ · Σ · w`.
- Все данные вынесены в `data/default.json`.
- Добавлены **индивидуальные ограничения** на минимальную и максимальную долю
  каждого актива (`min_weight`, `max_weight`).
- Ограничение полного вложения (сумма долей = 1) задано явно.
- Добавлена проверка согласованности размеров векторов и матрицы.

## Специфичные настройки
- Отдельных флагов нет — поведение определяется данными.
- В данных:
  - `returns`, `volatility` — векторы доходностей и рисков активов (в долях);
  - `correlation` — симметричная матрица корреляций с единицами на диагонали;
  - `min_weight`, `max_weight` — границы доли каждого актива.

## Файлы данных
- `data/default.json` — исходная задача из двух активов.
- `data/variation_three_assets.json` — портфель из трёх классов активов
  (акции/облигации/золото) без жёстких ограничений: показывает обобщение на
  большее число активов и эффект отрицательной корреляции.
- `data/variation_constrained.json` — тот же набор, но с индивидуальными
  границами долей: демонстрирует влияние ограничений на оптимум.

## Запуск
```bash
python task5_portfolio_optimization.py
python task5_portfolio_optimization.py --input data/variation_three_assets.json
python task5_portfolio_optimization.py --input data/variation_constrained.json
python task5_portfolio_optimization.py --interactive
```
