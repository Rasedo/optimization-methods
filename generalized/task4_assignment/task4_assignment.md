# Задача 4. Задача о назначениях

## Тип задачи
Задача о назначениях — частный случай целочисленного ЛП, решаемый венгерским
алгоритмом. Нужно взаимно-однозначно сопоставить исполнителей и работы так,
чтобы суммарная эффективность была максимальной (или суммарные затраты —
минимальными). Используется `linear_sum_assignment` из **SciPy**.

## Исходная постановка
Четыре сотрудника и четыре задачи, задана матрица эффективности «сотрудник ×
задача». Нужно назначить каждому сотруднику ровно одну задачу с максимальной
суммарной эффективностью.

## Как обобщено
- Поддерживается **произвольный размер** N×N.
- Матрица и подписи вынесены в `data/default.json`.
- Добавлено поле `sense`: `max` — максимизация эффективности (по умолчанию),
  `min` — минимизация затрат. Для максимизации матрица передаётся в алгоритм с
  обратным знаком.
- Исходный вариант содержал дублирующее решение через PuLP; в обобщённой версии
  оставлен единый компактный путь через венгерский алгоритм.
- Добавлена проверка квадратности и согласованности матрицы.

## Специфичные настройки
- В данных:
  - `sense` — `max` или `min`;
  - `efficiency` — квадратная матрица «исполнители × работы»;
  - `workers`, `tasks` — подписи строк и столбцов.

## Файлы данных
- `data/default.json` — исходная задача 4×4 на максимум эффективности.
- `data/variation_min_cost.json` — задача 3×3 на минимум затрат (`sense = min`):
  демонстрирует переключение цели.
- `data/variation_large.json` — задача 5×5: показывает масштабирование.

## Запуск
```bash
python task4_assignment.py
python task4_assignment.py --input data/variation_min_cost.json
python task4_assignment.py --input data/variation_large.json
python task4_assignment.py --interactive
```
